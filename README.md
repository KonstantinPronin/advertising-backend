# advertising-backend

#### Short description:
   A tiny service designed to storing and serving ads.   
   Default port 8080.
   
#### How to run:
```shell script
    git clone https://github.com/KonstantinPronin/advertising_backend.git
    cd advertising_backend
    docker-compose build
    docker-compose up
```
